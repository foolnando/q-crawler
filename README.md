#Usage

##training

`>> cd src`

`>> python classifier_util.py train`

##crawling
`>> cd src/spider`

`>> scrapy crawl apprentice %for the reinforcement learning based crawler`

`>> scrapy crawl baseline %for the reinforcement learning based crawler`

##Performance monitoring

`>> cd src/spider`

`>> make`
